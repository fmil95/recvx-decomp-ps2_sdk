EzADPCM - BGM (ADPCM waveform data) Playback with Disk Streaming

< Description of the Sample >
	This sample allows to reproduce the sound data of the ADPCM
	file with streaming from a disk.

	This sample is associated with the sample of the IOP and is
	functional in a pair.  Please refer to the readme file
	(ee/sample/sound/ezadpcm/readme_e.txt) provided for the EE
	sample.
