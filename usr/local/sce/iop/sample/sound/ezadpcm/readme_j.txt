EzADPCM - ディスクからの ADPCM データストリーミングによる BGM 再生

<サンプルの解説>
	ADPCM 波形データファイルをディスクからストリーミングして再生す
	るサンプルです。

	このサンプルは、EE 側のサンプルと対応しており、両方そろって機
        能します。詳しくは EE 側の readme
        (ee/sample/sound/ezadpcm/readme_j.txt) をご覧下さい。
