"PlayStation 2" Programmer Tool Runtime Library Release 2.0

関数リファレンス

このマニュアルでは、"PlayStation 2"開発環境で提供されているEEライブラリに含まれるデータ構造、 関数およびマクロについて解説しています。

このマニュアルの使い方

  • 左のフレームにはインデックスが示されます。インデックスには カテゴリー別一覧とABC順一覧があります。通常、上のフレームよりどちらかを選択して下さい。
  • インデックスより選択した項目の詳細がこのフレーム（右下）に表示されます。
  • フレームバーの位置はお好みの位置に調節することができます。マウスをバーの位置に合わせ、カーソルの形が変わったところでドラッグして下さい。
