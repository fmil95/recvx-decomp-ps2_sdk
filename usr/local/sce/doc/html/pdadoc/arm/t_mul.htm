ARM Architecture Reference Manual

MUL

機能説明

 Multiply

アーキテクチャ

 アーキテクチャ v4T のみ

Mul Rd, Rm

説明

MUL（Multiply）命令は、符号付きまたは符号なし変数を乗算して、32ビットの結果を生成します。

MULは、レジスタRmの値をレジスタRdの値で乗算し、結果をレジスタRdに戻します。（結果に基づいて）コンディション・コード・フラグが更新されます。

オペレーション

Rd = (Rm * Rd)[31:0]
N Flag = Rd[31]
Z Flag = if Rd == 0 then 1 else 0
C Flag = UNPREDICTABLE
V Flag = unaffected

例外

なし

修飾子

なし

注記

オペランドの制限：RdとRmに同じレジスタを指定すると、結果は予測できません。

アーリ・ターミネーション：乗算器インプリメンテーションが、アーリ・ターミネションをサポートしている場合、Rdオペランドの値にインプリメントしなければなりません。使用するアーリ・ターミネーションの種類（符号付きまたは符号なし）は、インプリメンテーションで定義されます。

符号付き、符号なし：MUL命令は64ビット積の下位32ビットしか生成しないため、符号付きの値、符号なしの値のいずれの乗算も同じ結果になります。
