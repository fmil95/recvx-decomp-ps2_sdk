ARM Architecture Reference Manual

LDRB（1）

機能説明

 Load unsigned byte immediate offset

アーキテクチャ

 アーキテクチャ v4T のみ

LDRB Rd, [Rn, #<5_bit_offset>]

説明

この形式のLDRB（Load Register Byte）命令は、8ビットのメモリ・データを汎用レジスタにロードし、その値を操作することができます。このアドレッシング・モードは、構造体（レコード）フィールドにアクセスするのに便利です。

オフセットがゼロの場合、生成されるアドレスはベース・レジスタRnの値と同じです。

この形式のLDRBは、以下の操作を行います。

  1. メモリから1バイトをロードし、
  2. このバイトを32ビット・ワードにゼロ拡張して、
  3. これをレジスタRdに書き込みます。

メモリ・アドレスは、<5_bit_offset>の値をレジスタRnの値に加算して計算されます。
