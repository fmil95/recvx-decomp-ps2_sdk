ARM Architecture Reference Manual

ORR

機能説明

 Logical OR

アーキテクチャ

 アーキテクチャ v4T のみ

ORR Rd, Rm

説明

ORR（Logical OR）命令は、レジスタの選択したビットをセットするのに使用します。各ビットについて、1とのORRはこのビットをセットし、0とのORRではビットは元のままです。

ORRは、レジスタRmの値とレジスタRdの値をビット単位で（包含的）ORを実行し、結果をレジスタRdに戻します。（結果に基づいて）コンディション・コード・フラグが変更されます。
