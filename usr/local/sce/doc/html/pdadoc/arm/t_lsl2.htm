ARM Architecture Reference Manual

LSL（2）

機能説明

 Logical shift left, register

アーキテクチャ

 アーキテクチャ v4T のみ

LSL Rd, Rs

説明

この形式のLSL（Logical Shift Left）命令は、レジスタの符号なしの値に（レジスタの）2の累乗変数を乗算した値を提供するのに使用します。

この場合、LSL命令はレジスタRdの値を、レジスタRsの最下位バイトの値によって論理左シフトし、結果をレジスタRdに戻します。空きビット位置にはゼロが挿入されます。（結果に基づいて）コンディション・コード・フラグが更新されます。

	C Flag = Rd[0]
	Rd = 0
else /* Rs[7:0] > 32 */

	Rd = 0
