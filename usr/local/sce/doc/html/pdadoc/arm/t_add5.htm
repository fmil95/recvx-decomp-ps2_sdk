ARM Architecture Reference Manual

ADD（5）

機能説明

 Add immediate to program counter

アーキテクチャ

 アーキテクチャ v4T のみ

ADD Rd, PC, #(<8_bit_immediate> * 4)

説明

この形式のADD命令は、PC相対（ワード・サイズの）変数をアドレス指定するのに使用します。

この場合、ADDはPCの値の下位2ビットをクリアして、8ビット・イミディエイト（0〜255）に加算し、結果をレジスタRdにストアします。
