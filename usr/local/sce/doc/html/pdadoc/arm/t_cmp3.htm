ARM Architecture Reference Manual

CMP（3）

機能説明

 Compare high registers

アーキテクチャ

 アーキテクチャ v4T のみ

CMP Rn, Rm

説明

この形式のCMP（Compare）命令は、上位レジスタの2つの算術値を比較します。

この場合、CMPは以下の比較を行います。

  • 下位レジスタと上位レジスタの値を比較する（H1=1、H2=0）、または
  • 上位レジスタと下位レジスタの値を比較する（H1=0、H2=1）、または
  • 上位レジスタと別の上位レジスタを比較し（H1=1、H2=1）、コンディション・コード・フラグをセットし、以降の命令が（条件分岐命令を使用して）条件付きで実行されるようにします。
