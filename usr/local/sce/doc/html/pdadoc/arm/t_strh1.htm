ARM Architecture Reference Manual

STRH（1）

機能説明

 Store halfword, immediate offset

アーキテクチャ

 アーキテクチャ v4T のみ

STRH Rd, [Rn, #<5_bit_offset> * 2]

説明

この形式のSTRH（Store Register Halfword）命令は、汎用レジスタの16ビットのデータをメモリにストアすることができます。このアドレッシング・モードは、構造体（レコード）フィールドにアクセスするのに便利です。オフセットがゼロの場合、生成されるアドレスはベース・レジスタRnの値と同じです。

この場合、STRHはレジスタRdの最下位ハーフワードの1ハーフワードをメモリにストアします。メモリ・アドレスは、<5_bit_offset>の値×2をレジスタRnの値に加算して計算されます。アドレスがハーフワードに整列していない場合、結果は予測できません。

注記
