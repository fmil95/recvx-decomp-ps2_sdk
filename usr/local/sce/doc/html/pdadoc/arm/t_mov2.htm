ARM Architecture Reference Manual

MOV（2）

機能説明

 Move high registers

アーキテクチャ

 アーキテクチャ v4T のみ

MOV Rd, Rm

説明

この形式のMOV（Move）命令は、上位レジスタとの間で値を移動するのに使用します。

この場合、MOVは以下の操作を行います。

  • 下位レジスタの値を上位レジスタに移動（H1=1、H2=0）、または
  • 上位レジスタの値を下位レジスタに移動（H1=0、H2=1）、または
  • 上位レジスタの値を別の上位レジスタに移動（H1=1、H2=1）します。

サブルーチン・リターン命令は、以下のとおりです。
MOV PC, LR
（BLシーケンスの実行後に行います。ＢＬ命令を参照）
