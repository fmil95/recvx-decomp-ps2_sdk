ARM Architecture Reference Manual

ADD（1）

機能説明

 Add immediate

アーキテクチャ

 アーキテクチャ v4T のみ

ADD Rd, Rn, #<3_bit_immediate>

説明

この形式のADD命令は、レジスタの値に小さな定数値を加算し、結果を第2レジスタにストアします。

この場合、ADDはレジスタRnの値と3ビット・イミディエイトの値（0〜7）を加算し、結果をデスティネーション・レジスタRdにストアします。（結果に基づいて）コンディション・コード・フラグが更新されます。
