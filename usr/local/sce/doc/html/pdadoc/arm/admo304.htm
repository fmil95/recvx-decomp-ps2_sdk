ARM Architecture Reference Manual

Addressing Mode 3

機能説明

 Register pre-indexed
アーキテクチャ アーキテクチャv4 のみ

[Rn, +/- Rm]!

説明

このアドレッシング・モードは、ベース・レジスタRnの値にインデックス・レジスタRmの値を加算するか、またはベース・レジスタRnの値からインデックス・レジスタRmの値を減算してアドレスを計算します。

命令で指定された条件がコンディション・コードのステータスと一致した場合、計算されたアドレスはベース・レジスタRnに書き戻されます。条件は、3.3「コンディション・フィールド」で定義されています。

オペレーション

if U == 1 then
        <address> = Rn + Rm
else /* U == 0 */
        <address> = Rn - Rm
if ConditionPassed(<cond>) then
        Rn = <address>

注記

Lビット：ロード命令(L==1)とストア命令(L==0)を区別します。

Sビット：符号付きハーフワード・アクセス(S==1)と符号なしハーフワード・アクセス(S==0)を区別します。

Hビット：ハーフワード・アクセス(H==1)と符号付きバイト・アクセス(H==0)を区別します。

R15の使用：レジスタRmまたはRnとしてR15を指定した場合、結果は予測できません。
