ARM Architecture Reference Manual

MOV（1）

機能説明

 Move immediate

アーキテクチャ

 アーキテクチャ v4T のみ

MOV Rd, #<8_bit_immediate>

説明

この形式のMOV（Move）命令は、大きな定数値をレジスタに移動します。
この場合、MOVはデスティネーション・レジスタRdに8ビット・イミディエイトの値（0〜255）を書き込みます。（結果に基づいて）コンディション・コード・フラグが更新されます。
