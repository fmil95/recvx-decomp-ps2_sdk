ARM Architecture Reference Manual

LDR（2）

機能説明

 Load word register offset

アーキテクチャ

 アーキテクチャ v4T のみ

LDR Rd, [Rn, Rm]

説明

この形式のLDR（Load Register）命令は、32ビットのメモリ・データを汎用レジスタにロードし、その値を操作することができます。
このアドレッシング・モードは、ポインタ＋大きなオフセットの算術（MOVイミディエイトを使用して、オフセットを設定）や配列の1つの要素にアクセスするのに便利です。

この場合、LDRはメモリから1ワードをロードし、それをレジスタRdに書き込みます。メモリ・アドレスは、レジスタRmの値をレジスタRnの値に加算して計算されます。アドレスがワードに整列していない場合、結果は予測できません。

注記

アラインメント：インプリメンテーションに、システム・コントロール・コプロセッサ（英文リファレンスマニュアル第7章「システム・アーキテクチャとシステム・コントロール・コプロセッサ」を参照）が含まれており、アラインメント・チェックがイネーブルされている場合、ビット[1:0]!=0b00のアドレスによって、アラインメント例外が発生します。
