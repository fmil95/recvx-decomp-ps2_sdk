ARM Architecture Reference Manual

STRB（1）

機能説明

 Store byte immediate offset

アーキテクチャ

 アーキテクチャ v4T のみ

STRB Rd, [Rn, #<5_bit_offset>]

説明

この形式のSTRB（Store Register Byte）命令は、汎用レジスタの8ビットのデータをメモリにストアすることができます。このアドレッシング・モードは、構造体（レコード）フィールドにアクセスするのに便利です。オフセットがゼロの場合、生成されるアドレスはベース・レジスタRnの値と同じです。

この場合、STRBはレジスタRdの最下位バイトから1バイトをメモリにストアします。メモリ・アドレスは、<5_bit_offset>の値をレジスタRnの値に加算して計算されます。
