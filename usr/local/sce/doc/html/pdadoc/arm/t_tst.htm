ARM Architecture Reference Manual

TST

機能説明

 Test bits

アーキテクチャ

 アーキテクチャ v4T のみ

TST Rn, Rm

説明

TST（Test）命令は、レジスタの多数のビットがすべてクリアされているか、またはレジスタの最低1ビットがセットされているかを判定するのに使用します。

TSTは、レジスタRmの値とレジスタRdの値を論理ANDして比較します。（結果に基づいて）コンディション・コード・フラグが更新されます。
