ARM Architecture Reference Manual

CMN

機能説明

 Compare negative (register)

アーキテクチャ

 アーキテクチャ v4T のみ

CMN Rn, Rm

説明

CMN（Compare Negative）命令は、算術値と算術値の負数を比較し、以降の命令が（条件分岐命令を使用して）条件付きで実行されるよう、コンディション・コード・フラグをセットします。

CMNは、レジスタRmの値とレジスタRdの値を加算（またはこの値の負数を減算）して比較を行います。（結果に基づいて）コンディション・コード・フラグが更新されます。
