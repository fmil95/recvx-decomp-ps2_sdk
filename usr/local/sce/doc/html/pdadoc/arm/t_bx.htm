ARM Architecture Reference Manual

BX

機能説明

 Branch with exchange

アーキテクチャ

 アーキテクチャ v4T のみ

BX Rm

説明

BX（Branch and Exchange）命令は、ARMコードとThumbコードの間で分岐するのに使用します。

BXは分岐し、分岐先で命令をデコードするのに使用する命令セット・デコーダを選択します。分岐先アドレスはレジスタRmの値です。Tフラグは、レジスタRmの値のビット0で更新されます。

コンディション・コード

注記

H2ビット：このビットは以下のような上位レジスタ指示子です。
H2==0 Rmが下位レジスタを指定することを示します。
H2==1 Rmが上位レジスタを指定することを示します。

Thumbへの転送：Thumb命令セットに転送する場合、Rmのビット[0]はPCに転送されるとゼロにセットされます。

ARMへの転送：ARM命令セットに転送する場合、Rmのビット[1:0]はPCに転送されるとゼロにセットされます。
