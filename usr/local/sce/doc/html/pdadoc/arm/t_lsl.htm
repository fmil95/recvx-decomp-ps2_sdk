ARM Architecture Reference Manual

LSL（1）

機能説明

 Logical shift left, immediate

アーキテクチャ

 アーキテクチャ v4T のみ

LSL Rd, Rm, #<shift_imm>

説明

この形式のLSL（Logical Shift Left）命令は、レジスタの値を直接（LSL #0）、またはレジスタの値に2の累乗定数を乗算した値を提供するのに使用します。

この場合、LSLはレジスタRmの値を、0〜31の範囲のイミディエイト値によって論理左シフトし、結果をデスティネーション・レジスタRdにストアします。空きビット位置にはゼロが挿入されます。（結果に基づいて）コンディション・コード・フラグが更新されます。
