ARM Architecture Reference Manual

3.1 この章の使い方

この章は以下の項目に分かれています。

1. ARM命令型の概要
2. 命令のアルファベット順リスト
3. アドレッシング・モード

3.1.1 ARM命令型の概要

ここでは命令セットを機能別に分類し、関連する使用例とエンコーディングを示します。また、各機能グループにはすべての命令がリストされており、これらの命令はアルファベット順リストにも記載されています。機能グループは以下のとおりです。

1. 分岐
2. データ処理
3. 乗算
4. ステータス・レジスタ・アクセス
5. 以下のロード/ストア
− ワードまたは符号なしバイトのロード/ストア
− ハーフワードのロード/ストアと符号付きバイトのロード
− 複数ロード/ストア
6. セマフォ
7. コプロセッサ

3.1.2 命令のアルファベット順リスト

ここではすべてのARM命令をリストします。また、以下の情報を示します。

  • 命令の構文と機能グループ
  • エンコーディングと動作内容
  • 関連の例外と修飾子
  • 使用に関する注記
  • ARMアーキテクチャの可用性に関する制限
  • 関連のアドレッシング・モードに対するクロス・リファレンス

3.1.3 アドレッシング・モード

ここでは機能別に分類された命令のアドレッシング・モードをリストします。

モード1 データ処理命令のシフタ・オペランド
モード2 ワードまたは符号なしバイトのロード/ストア・アドレッシング・モード
モード3 ハーフワードのロード/ストア、または符号付きバイトのロード・アドレッシングモード
モード4 複数ロード/ストア・アドレッシング・モード
モード5 コプロセッサのロード/ストア・アドレッシング・モード
