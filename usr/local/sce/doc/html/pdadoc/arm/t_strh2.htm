ARM Architecture Reference Manual

STRH（2）

機能説明

 Store halfword, register offset

アーキテクチャ

 アーキテクチャ v4T のみ

STRH Rd, [Rn, Rm]

説明

この形式のSTRH（Store Register Halfword）命令は、汎用レジスタの16ビットのデータをメモリにストアすることができます。このアドレッシング・モードは、ポインタ＋大きなオフセットの算術（MOVイミディエイトを使用してオフセットを設定）や配列の1要素にアクセスするのに便利です。

この場合、STRHはレジスタRdの最下位ハーフワードの1ハーフワードをメモリにストアします。メモリ・アドレスは、レジスタRmの値をレジスタRnの値に加算して計算されます。アドレスがハーフワードに整列していない場合、結果は予測できません。

注記
