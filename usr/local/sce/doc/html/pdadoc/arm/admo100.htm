ARM Architecture Reference Manual

Addressing Mode 1

機能説明

 Immediate

#<immediate>

説明

<shifter_operand>値は、8ビット・イミディエイト値を、32ビット・ワードの任意の偶数ビット位置まで（右に）ローテートして形成されます。ローテート・イミディエイトがゼロの場合、シフタからのキャリアウトはCフラグの値になります。そうでない場合は、<shifter_operand>の値のビット31がセットされます。

データ処理オペランドは、データ処理命令に定数（命令で定義される）オペランドを提供します。

31 - - 28 27 26 25 24 - - 21 20 19 - - 16 15 - - 12 11 - - 8 7 - - - - - - 0


cond


0


0


1


opcode


S


Rn


Rd


rotate_imm


8_bit_immediate

オペレーション

<shifter_operand> = <8_bit_immediate> Rotate_Right (<rotate_imm> * 2)
if <rotate_imm> == 0 then
        <shifter_carry_out> = C flag
else /* <rotate_imm> != 0 */
        <shifter_carry_out> = <shifter_operand>[31]

注記

適合イミディエイト：32ビットのイミディエイトすべてが適合するわけではありません。8ビット・イミディエイトを偶数量だけ右にローテートして形成できるものだけが、このフォーマットの有効な32ビット・イミディエイトです。

別のアセンブラ表記：32ビット・イミディエイトは、以下によっても指定できます。

#<8_bit_immediate>, <rotate_amount>

ここで、

	<rotate_amount> = <rotate_imm> << 1
