ARM Architecture Reference Manual

ADD（4）

機能説明

 Add high register

アーキテクチャ

 アーキテクチャ v4T のみ

ADD Rd, Rm

説明

この形式のADD命令は上位レジスタの値の加算に使用します。

この場合、ADDは以下の加算を実行します。

  • 下位レジスタの値を上位レジスタの値に加算（H1=1、H2=0）する。または、
  • 上位レジスタの値を下位レジスタの値に加算（H1=0、H2=1）する。または、
  • 上位レジスタの値をもう1つの上位レジスタに加算（H1=1、H2=1）する。

コンディション・コード・フラグに影響はありません。
