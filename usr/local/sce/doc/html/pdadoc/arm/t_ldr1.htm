ARM Architecture Reference Manual

LDR（1）

機能説明

 Load word immediate offset

アーキテクチャ

 アーキテクチャ v4T のみ

LDR Rd, [Rn, #<5_bit_offset> * 4]

説明

この形式のLDR（Load Register）命令では、32ビットのメモリ・データを汎用レジスタにロードし、その値を操作することができます。

このアドレッシング・モードは、構造体（レコード）フィールドにアクセスするのに便利です。オフセットがゼロの場合、生成されるアドレスはベース・レジスタRnの値と同じです。

この場合、LDRはメモリから1ワードをロードし、それをレジスタRdに書き込みます。メモリ・アドレスは、<5_bit_offset>の値×4をレジスタRnの値に加算して計算されます。アドレスがワードに整列していない場合、結果は予測できません。
