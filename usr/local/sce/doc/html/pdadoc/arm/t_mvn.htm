ARM Architecture Reference Manual

MVN

機能説明

 Move NOT (register)

アーキテクチャ

 アーキテクチャ v4T のみ

MVN Rd, Rm

説明

MVN（Move NOT）命令はレジスタ値の補数をとるのに使用します。ビット・マスクの形成によく用いられます。

MVNは、レジスタRmの1の論理補数値をデスティネーション・レジスタRdに書き込みます。（結果に基づいて）コンディション・コード・フラグが更新されます。
