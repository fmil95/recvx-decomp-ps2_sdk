ARM Architecture Reference Manual

CMP（1）

機能説明

 Compare immediate

アーキテクチャ

 アーキテクチャ v4T のみ

CMP Rn, #<8_bit_immediate>

説明

この形式のCMP（Compare）命令は、2つの算術値を比較してコンディション・コード・フラグをセットします。これにより、以降の命令は（条件分岐命令を使用して）条件付きで実行されます。

この場合、CMPはレジスタRdの値から8ビット・イミディエイトの値（0〜255）を減算して比較を実行します。（結果に基づいて）コンディション・コード・フラグが更新されます。
