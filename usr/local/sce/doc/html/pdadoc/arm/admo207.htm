ARM Architecture Reference Manual

Addressing Mode 2

機能説明

 Immediate post-indexed

[Rn], #+/-<12_bit_offset>

説明

このアドレッシング・モードは、ポインタ値の自動更新を伴う配列へのポインタ・アクセスに使用します。

ベース・レジスタRnの値からアドレスを計算します。

命令で指定された条件がコンディション・コードのステータスと一致した場合、ベース・レジスタRnの値にイミディエイト・オフセットの値が加算されるか、またはベース・レジスタRnの値からイミディエイト・オフセットの値が減算されベース・レジスタRnに書き戻されます。条件は、3.3「コンディション・フィールド」で定義されています。

注記

Bビット：符号なしバイト・アクセス（B==1）とワード・アクセス（B==0）を区別します。

Lビット：ロード(L==1)命令とストア(L==0)命令を区別します。

R15の使用：レジスタRnとしてR15を指定した場合、結果は予測できません。
