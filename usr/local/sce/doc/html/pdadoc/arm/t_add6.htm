ARM Architecture Reference Manual

ADD（6）

機能説明

 Add immediate to stack pointer

アーキテクチャ

 アーキテクチャ v4T のみ

ADD Rd, SP, #(<8_bit_immediate> * 4)

説明

この形式のADD命令は、SP相対（ワード・サイズの）変数をアドレス指定するのに使用します。

この場合、ADDはSPの値と8ビット・イミディエイトの値（0〜255）を加算し、結果をレジスタRdにストアします。
