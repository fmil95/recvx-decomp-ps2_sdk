ARM Architecture Reference Manual

SUB（2）

機能説明

 Subtract large immediate

アーキテクチャ

 アーキテクチャ v4T のみ

SUB Rd, #<8_bit_immediate>

説明

この形式のSUB（Subtract）命令は、レジスタの値から大きな定数値を減算し、結果を同じレジスタに戻します。

この場合、SUBはレジスタRdの値から8ビット・イミディエイトの値（0〜255）を減算し、結果をレジスタRdに戻します。（結果に基づいて）コンディション・コード・フラグが更新されます。

Rd = Rd - <8_bit_immed>

V Flag = OverflowFrom(Rd - <8_bit_immed>)
