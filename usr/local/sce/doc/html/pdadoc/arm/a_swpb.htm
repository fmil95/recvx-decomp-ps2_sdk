ARM Architecture Reference Manual

SWPB

機能説明

 セマフォ
アーキテクチャ アーキテクチャｖ１、ｖ２ではリポートされない

SWP{<cond>}B Rd, Rm, [Rn]

説明

SWPB (Swap Byte)命令は、レジスタとメモリ間で1バイトを交換します。

SWPBは、レジスタRnの値で与えられるメモリ・アドレスから1バイトをロードします。レジスタRmの最下位バイトの値は、Rnで与えられるメモリ・アドレスにストアされ、それのアドレスにロードされていた値は32ビット・ワードにゼロ拡張され、レジスタRdに書き込まれます。RnとRdに同じレジスタを指定した場合、レジスタの最下位バイトの値とメモリ・アドレスのバイト値をスワップします。

この命令は、命令で指定された条件がコンディション・コードのステータスと一致した場合にのみ実行されます。条件は、3.3「コンディション・フィールド」で定義されています。

注記

R15の使用：Rd、Rn、またはRmにレジスタ15を指定した場合、結果は予測できません。

オペランドの制限：RnとRm、またはRnとRdに同じレジスタを指定した場合、結果は予測できません。

データ・アボート：ロード・アクセスまたはストア・アクセスのいずれか（または両方）でデータ・アボートが通知された場合、ロードされた値はRdに書き込まれません。
