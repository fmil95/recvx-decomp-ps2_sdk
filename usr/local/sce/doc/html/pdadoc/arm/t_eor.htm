ARM Architecture Reference Manual

EOR

機能説明

 Exclusive OR

アーキテクチャ

 アーキテクチャ v4T のみ

EOR Rd, Rm

説明

  • EOR（Exclusive OR）命令は、レジスタの選択したビットを反転するのに使用します。各ビットについて、1と指定したビットを反転し、0と指定したビットは元のまま残します。
  • EORはレジスタRmの値とレジスタRdの値で、ビット単位の排他的ORを実行し、レジスタRdに結果を戻します。（結果に基づいて）コンディション・コード・フラグが更新されます。
