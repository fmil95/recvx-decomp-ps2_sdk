Addressing Mode 1

機能説明

 Register

Rm

説明

このデータ処理オペランドは、レジスタの値を直接提供します。

これはレジスタRmの値で生成される命令オペランドです。シフタからCフラグがキャリアウトされます。

オペレーション

<shifter_operand> = Rm
<shifter_carry_out> = C Flag

注記

エンコーディング：この命令は、シフトがゼロ（<shift_imm> == 0）のイミディエイトによる論理右シフト（次ページ参照）としてエンコードされます。

R15の使用：レジスタRmまたはRnとしてR15を指定した場合、使用される値は現在の命令のアドレスに8を加算した値です。
