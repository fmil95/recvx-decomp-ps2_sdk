ARM Architecture Reference Manual

Addressing Mode 2

機能説明

 Register offset

[Rn, #+/-<12_bit_offset]

説明

このアドレッシング・モードは、ポインタ＋オフセット演算および配列の1要素へのアクセスに使用します。

インデックス・レジスタRmの値をベース・レジスタRnの値に加算するか、またはベース・レジスタRnの値からインデックス・レジスタRmの値を減算することによってアドレスを計算します。

オペレーション

if U == 1 then
        <address> = Rn + Rm
else /* U == 0 */
        <address> = Rn - Rm

注記

エンコーディング：このアドレッシング・モードは、ゼロにスケールされた、LSLスケール付きレジスタ・オフセットとしてエンコードされます。

Bビット：符号なしバイト・アクセス（B==1）とワード・アクセス（B==0）を区別します。

Lビット：ロード(L==1)命令とストア(L==0)命令を区別します。

R15の使用：レジスタRnとしてR15を指定した場合、使用される値は命令のアドレスに8を加算した値です。レジスタRmとしてR15を指定した場合、結果は予測できません。
