ARM Architecture Reference Manual

1.4 分岐命令

データ処理命令またはロード命令が（プログラム・カウンタに書き込むことによって）制御フローを変更できるのと同様、標準分岐命令には、最大32メガバイトの前方および後方分岐を可能にする24ビットの符号付きオフセットがあります。

分岐後にR14（リンク・レジスタ；LR）に命令のアドレスも保持するBranch with Linkオプションがあり、このオプションを使用するとMOVE命令によってLRの内容をPCに置き、分岐後にその命令に戻ることができるため、サブルーチン・コールを行うことができます。

また、ソフトウェア割り込み（SWI）と呼ばれる特殊な分岐命令もあります。この分岐命令は、オペレーティング・システムの呼び出し（OS定義のサービスを要求する呼び出し）を行います。SWIもプロセッサ・モードを変更することにより、非特権タスクにOS特権（この特権へのアクセスはOSによって制御されます）を与えることができます。

THUMB命令セットをインプリメントするプロセッサでは、レジスタで指定されたアドレスにジャンプし、かつオプションとしてスイッチ命令セットを切り換える分岐命令があり、ARMコードからTHUMBコードを、またTHUMBコードからARMコードを呼び出すことができます。THUMB命令の概要については、第6章「Thumb命令セット」を参照してください。
