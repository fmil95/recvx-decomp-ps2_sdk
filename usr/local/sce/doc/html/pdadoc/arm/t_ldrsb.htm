ARM Architecture Reference Manual

LDRSB

機能説明

 Load signed byte, register offset

アーキテクチャ

 アーキテクチャ v4T のみ

LDRSB Rd, [Rn, Rm]

説明

LDRSB（Load Register Signed Byte）命令は、8ビットの符号付きメモリ・データを汎用レジスタにロードし、その値を操作することができます。このアドレッシング・モードは、ポインタ＋大きなオフセットの算術（MOVイミディエイトを使用してオフセットを設定）や配列の1つの要素にアクセスするのに便利です。

LDRSBは、以下の操作を実行します。

  1. メモリから1バイトをロードし、
  2. このバイトを32ビット・ワードに符号拡張して、
  3. これをレジスタRdに書き込みます。

メモリ・アドレスは、レジスタRmの値をレジスタRnの値に加算して計算されます。
