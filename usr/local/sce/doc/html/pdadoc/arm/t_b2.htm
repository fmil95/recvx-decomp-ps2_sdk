ARM Architecture Reference Manual

B（2）

機能説明

 Unconditional branch

アーキテクチャ

 アーキテクチャ v4T のみ

B <target_address>

説明

この形式のB（分岐）命令は、プログラム・フローを無条件で変化させます。

この場合、Bによってターゲット・アドレスへの無条件分岐が発生します。分岐先アドレスは、11ビットの符号付き（2の補数）オフセットを左に1ビットシフトし、結果を32ビットに符号拡張して、これをPCの内容（分岐命令のアドレス+4が収められている）に加算することによって計算します。したがって、この命令は+/-2048バイトの分岐を指定することができます。

注記

オフセット計算：アセンブラは、現在の命令のアドレスと分岐先のアドレス（プログラム・ラベルとして与えられたもの）の差から4を減算して（PCは現在の命令のアドレス＋4を保持しているため）、分岐オフセットのアドレスを計算します。

メモリ境界：ロケーション・ゼロを超えて後方に分岐したり、32ビット・アドレス空間の上限を超えて分岐した場合、結果は予測できません。
