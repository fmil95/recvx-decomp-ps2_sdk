ARM Architecture Reference Manual

ADD（2）

機能説明

 Add large immediate

アーキテクチャ

 アーキテクチャ v4T のみ

ADD Rd, #<8_bit_immediate>

説明

この形式のADD命令もレジスタの値に大きな定数値を加算し、結果を同じレジスタに戻すのに使用します。

この場合、ADD命令はレジスタRdの値と8ビット・イミディエイトの値（0〜255）を加算し、結果をレジスタRdに戻します。（結果に基づいて）コンディション・コード・フラグが更新されます。
