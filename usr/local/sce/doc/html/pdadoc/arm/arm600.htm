ARM Architecture Reference Manual
６ Thumb命令セット

Thumb命令セットは、ARM命令セットのサブセットです。

Thumbは、メモリ・データ・バスを使用するARMインプリメンテーションのパフォーマンスを向上させ、ARM以上のコード密度が可能になるように設計されています。ARMv4Tアーキテクチャは、フル32ビットARM命令セットと16ビットThumb命令セットの両方を内蔵しています。Thumbの各命令は、16ビットでエンコードします。

この章では、Thumbの各命令を示し、フォーマットとコード化に関する情報を提供します。
