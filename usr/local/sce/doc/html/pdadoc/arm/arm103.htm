ARM Architecture Reference Manual

1.3 ARM命令セット

ARM命令セットは、次の4つクラスに大別できます。

  • 分岐
  • データ処理
  • ロードとストア
  • コプロセッサ
条件付き実行

ARM命令はすべて条件付きで実行できます。データ処理命令（および一部のコプロセッサ命令）では、実行結果に応じて、CPSRの4つのコンディション・コード・フラグ（ネガティブ、ゼロ、キャリー、オーバーフロー）を更新することができます。後続の命令は、コンディション・コード・フラグの状態に応じて、条件付きで実行できます。コンディション・コード・フラグの値によって15の条件がインプリメントされますが、通常の（無条件）命令が常に実行されるように、1つの条件は実際にはコンディション・コード・フラグを無視します。
