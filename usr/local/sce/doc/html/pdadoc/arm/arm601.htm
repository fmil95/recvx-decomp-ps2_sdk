ARM Architecture Reference Manual

6.1 この章の使い方

この章は以下の3つの部分に分かれています。

  1. Thumbについて
  2. Thumb命令タイプの概要
  3. 命令のアルファベット順リスト

6.1.1 Thumbについて（セクション6.2および6.3）

ここでは、Thumbの概念とARM命令の実行との関連性について説明します。

6.1.2 Thumb命令タイプの概要（セクション6.4〜6.7）

ここでは、Thumb命令セット内の機能グループを説明し、関連する例とエンコーディングを示します。各機能グループにはすべての命令がリストされており、これを使ってアルファベット順のセクションで命令を検索することができます。機能グループは、以下のとおりです。

  1. 分岐命令
  2. データ処理命令
  3. レジスタのロードおよびストア命令
  4. ロードおよびストア・マルチプル命令

6.1.3 命令のアルファベット順リスト（セクション6.8）

ここでは、Thumbの各命令をアルファベット順にリストし、以下を提示します。

  • 命令の構文と機能グループ
  • エンコーディングとオペレーション
  • 関連の例外と修飾子
  • 使い方の注記

関連箇所で、命令の説明には、同等なARM命令とエンコーディングを示します。
