ARM Architecture Reference Manual

LDRB（2）

機能説明

 Load unsigned byte, register offset

アーキテクチャ

 アーキテクチャ v4T のみ

LDRB Rd, [Rn, Rm]

説明

この形式のLDRB（Load Register Byte）命令は、8ビットのメモリ・データを汎用レジスタにロードし、その値を操作することができます。このアドレッシング・モードは、ポインタ＋大きなオフセットの算術（MOVイミディエイトを使用してオフセットを設定）や配列の1つの要素にアクセスするのに便利です。

この場合、LDRBは以下の操作を行います。

  1. メモリから1バイトをロードし、
  2. このバイトを32ビット・ワードにゼロ拡張して、
  3. これをレジスタRdに書き込みます。

メモリ・アドレスは、レジスタRmの値をレジスタRnの値に加算して計算されます。
