ARM Architecture Reference Manual

Addressing Mode 1

機能説明

 Rotate right by immediate

Rm, ROR #<shift_imm>

説明

このデータ処理オペランドは、レジスタの値を定数値分だけローテートするのに使用します。

命令のオペランドは、レジスタRmの値を1〜31の範囲のイミディエイト値で右にローテートして生成されます。右端からローテート・オフされたビットが左の空きビット位置に挿入されます。

<shift_imm> == 0のときは、拡張付き右ローテートが実行されます（Rm, RRX参照）。

オペレーション

if <shift_imm> == 0 then
        See Rm, RRX
else /* <shift_imm> > 0 */
        <shifter_operand> = Rm Rotate_Right <shift_imm>
        <shifter_carry_out> = Rm[<shift_imm> - 1]

注記

R15の使用：レジスタRmまたはRnとしてR15を指定した場合、使用される値は現在の命令のアドレスに8を加算した値です。
