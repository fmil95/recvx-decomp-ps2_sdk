ARM Architecture Reference Manual

ADD（3）

機能説明

 Add register

アーキテクチャ

 アーキテクチャ v4T のみ

ADD Rd, Rm

説明

この形式のADD命令は、あるレジスタの値を第2のレジスタの値に加算して、結果を3番目のレジスタにストアします。

この場合、ADDはレジスタRnの値とレジスタRmの値を加算し、結果をデスティネーション・レジスタRdにストアします。（結果に基づいて）コンディション・コード・フラグが更新されます。
