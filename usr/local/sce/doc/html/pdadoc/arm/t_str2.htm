ARM Architecture Reference Manual

STR（2）

機能説明

 Store word, register offset

アーキテクチャ

 アーキテクチャ v4T のみ

STR Rd, [Rn, Rm]

説明

この形式のSTR（Store Register）命令は、汎用レジスタの32ビットのデータをメモリにストアすることができます。このアドレッシング・モードは、ポインタ＋大きなオフセットの算術（MOVイミディエイトを使用してオフセットを設定）や配列の1つの要素にアクセスするのに便利です。

この場合、STRはレジスタRdの1ワードをメモリにストアします。メモリ・アドレスは、レジスタRmの値にレジスタRnの値を加算して計算されます。アドレスがワードに整列していない場合、結果は予測できません。

注記

アラインメント：インプリメンテーションに、システム・コントロール・コプロセッサ（英文リファレンスマニュアル第7章「システム・アーキテクチャとシステム・コントロール・コプロセッサ」を参照）が含まれており、アラインメント・チェックがイネーブルされている場合、ビット[1:0]!=0b00のアドレスによって、アラインメント例外が発生します。
