ARM Architecture Reference Manual

SUB（3）

機能説明

 Subtract register

アーキテクチャ

 アーキテクチャ v4T のみ

SUB Rd, Rn, Rm

説明

この形式のSUB（Subtract）命令は、あるレジスタの値を別のレジスタの値から減算し、結果を第3のレジスタにストアします。

この場合、SUBはレジスタRnの値からレジスタRmの値を減算し、結果をデスティネーション・レジスタRdにストアします。（結果に基づいて）コンディション・コード・フラグが更新されます。
