ARM Architecture Reference Manual

Addressing Mode 3

機能説明

 Register offset
アーキテクチャ アーキテクチャv4 のみ

[Rn, +/- Rm]

説明

このアドレッシング・モードは、ポインタ＋オフセット演算および配列の1要素へのアクセスに便利です。

ベース・レジスタRnの値にインデックス・レジスタRmの値を加算するか、またはベース・レジスタRnの値からインデックス・レジスタRmの値を減算することによってアドレスを計算します。

オペレーション

if U == 1 then
        <address> = Rn + Rm
else /* U == 0 */
        <address> = Rn - Rm

注記

Lビット：ロード命令(L==1)とストア命令(L==0)を区別します。

Sビット：符号付きハーフワード・アクセス(S==1)と符号なしハーフワード・アクセス(S==0)を区別します。

Hビット：ハーフワード・アクセス(H==1)と符号付きバイト・アクセス(H==0)を区別します。

R15の使用：レジスタRnとしてR15を指定した場合、使用される値は命令のアドレスに8を加算した値です。レジスタRmとしてR15を指定した場合、結果は予測できません。
