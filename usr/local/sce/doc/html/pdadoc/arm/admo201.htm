ARM Architecture Reference Manual

Addressing Mode 2

機能説明

 Immediate offset

[Rn, #+/-<12_bit_offset]

説明

このアドレッシング・モードは、構造体（レコード）フィールドへのアクセス、およびスタック・フレーム内のパラメータとローカル変数へのアクセスに便利です。オフセットがゼロの場合、生成されるアドレスはベース・レジスタRnの値です。

ベース・レジスタRnの値にイミディエイト・オフセットの値を加算するか、またはベース・レジスタRnからイミディエイト・オフセットの値を減算してアドレスを計算します。

オペレーション

if U == 1 then
        <address> = Rn + <12_bit_offset>
else /* U == 0 */
        <address> = Rn - <12_bit_offset>

注記

Bビット：符号なしバイト・アクセス（B==1）とワード・アクセス（B==0）を区別します。

Lビット：ロード(L==1)とストア(L==0)を区別します。

R15の使用：レジスタRnとしてR15を指定した場合、使用される値は命令のアドレスに8を加算した値です。
