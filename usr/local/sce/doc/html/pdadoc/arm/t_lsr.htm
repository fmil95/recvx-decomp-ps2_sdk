ARM Architecture Reference Manual

LSR（1）

機能説明

 Logical shift right, immediate

アーキテクチャ

 アーキテクチャ v4T のみ

LSR Rd, Rm, #<shift_imm>

説明

この形式のLSR（Logical Shift Right）命令は、レジスタの値を2の累乗定数で除算するのに使用します。

この場合、LSRはレジスタRmの値を、1〜32の範囲のイミディエイト値によって論理右シフトし、結果をデスティネーション・レジスタRdにストアします。空きビット位置にはゼロが挿入されます。

32によるシフトは、以下のようにエンコードされます。
<shift_imm> = 0

（結果に基づいて）コンディション・コード・フラグが更新されます。
