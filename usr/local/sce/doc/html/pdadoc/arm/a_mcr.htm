ARM Architecture Reference Manual

MCR

機能説明

 コプロセッサ

アーキテクチャ

 アーキテクチャ v１ではサポートされない

MCR{<cond>} p<cp_num>, <opcode_1>, Rd, CRn, CRm, <opcode_2>

説明

MCR（Move to Coprosessor from ARM Register）命令は、浮動小数点アクセラレータ・コプロセッサの固定小数点から浮動小数点への変換命令など、ARMレジスタの値を操作するコプロセッサ命令を開始するのに使用します。

MCRは<cp_num>によって指定されたコプロセッサに、レジスタRdの値を渡します。この命令を実行できることを示すコプロセッサがない場合、未定義命令例外が発生します。

この命令は、命令で指定された条件がコンディション・コードのステータスと一致した場合にのみ実行されます。条件は、3.3「コンディション・フィールド」で定義されています。
