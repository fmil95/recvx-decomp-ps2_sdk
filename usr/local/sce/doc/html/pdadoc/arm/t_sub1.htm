ARM Architecture Reference Manual

SUB（1）

機能説明

 Subtract immediate

アーキテクチャ

 アーキテクチャ v4T のみ

SUB Rd, Rn, #<3_bit_immediate>

説明

この形式のSUB（Subtract）命令は、レジスタの値から小さな定数値を減算し、結果を第2レジスタにストアします。

この場合、SUBはレジスタRnの値から3ビット・イミディエイトの値（0〜7）を減算し、結果をデスティネーション・レジスタRdにストアします。（結果に基づいて）コンディション・コード・フラグが更新されます。
