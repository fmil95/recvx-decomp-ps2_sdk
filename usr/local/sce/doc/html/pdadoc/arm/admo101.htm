ARM Architecture Reference Manual

Addressing Mode 1

機能説明

 Logical shift left by immediate

Rm, LSL #<shift_imm>

説明

このデータ処理オペランドは、レジスタの値を直接（ローン・レジスタ・オペランド：前ページ参照）または左シフト（2の累乗定数で乗算）した値を提供するのに使用します。

この命令オペランドは、レジスタRmの値を0〜31の範囲のイミディエイト値で論理左シフトして生成されます。空きビット位置にはゼロが挿入されます。シフタからのキャリアウトが最後にシフトアウトされるが、シフトが指定されていない場合（ローン・レジスタ・オペランド：前ページ参照）は、Cフラグが最後のビットになります。

オペレーション

if <shift_imm> == 0 then /* Register Operand */
        <shifter_operand> = Rm
        <shifter_carry_out> = C Flag
else /* <shift_imm> > 0 */
        <shifter_operand> = Rm Logical_Shift_Left <shift_imm>
        <shifter_carry_out> = Rm[32 - <shift_imm>]

注記

デフォルト・シフト：<shift_imm> == 0の場合は、オペランドをRmとして記述することができます（前ページ参照）。

R15の使用：レジスタRmまたはRnとしてR15を指定した場合、使用される値は現在の命令のアドレスに8を加算した値です。
