ARM Architecture Reference Manual

Addressing Mode 2

機能説明

 Immediate pre-indexed

[Rn, #+/-<12_bit_offset>]!

説明

このアドレッシング・モードは、ポインタ値の自動更新を伴う配列へのポインタ・アクセスに使用します。

ベース・レジスタRnの値にイミディエイト・オフセットの値を加算するか、またはベース・レジスタRnからイミディエイト・オフセットの値を減算してアドレスを計算します。

命令で指定された条件がコンディション・コードのステータスと一致した場合、計算されたアドレスはベース・レジスタRnに書き戻されます。条件は、3.3「コンディション・フィールド」で定義されています。

オペレーション

if U == 1 then
        <address> = Rn + <12_bit_offset>
else /* if U == 0 */
        <address> = Rn - <12_bit_offset>
if ConditionPassed(<cond>) then
        Rn = <address>

注記

Bビット：符号なしバイト・アクセス（B==1）とワード・アクセス（B==0）を区別します。

Lビット：ロード(L==1)命令とストア(L==0)命令を区別します。

R15の使用：レジスタRnとしてR15を指定した場合、結果は予測できません。
