ARM Architecture Reference Manual

ROR

機能説明

 Rotate right (register)

アーキテクチャ

 アーキテクチャ v4T のみ

ROR Rd, Rs

説明

ROR（Rotate Right Register）命令は、レジスタの値を（レジスタの）変数値でローテートするのに使用します。

RORはレジスタRdの値を、レジスタRsの最下位バイトの値によって、右にローテートし、結果をレジスタRdに戻します。右端からローテート・オフしたビットは、左の空きビット位置に挿入されます。（結果に基づいて）コンディション・コード・フラグが更新されます。
