ARM Architecture Reference Manual

BX

機能説明

 分岐

アーキテクチャ

 アーキテクチャ v4T のみ

BX{<cond>} Rm

説明

BX（Branch and Exchange命令セット）は、ARMアーキテクチャ・バージョン4では未定義です。ARMアーキテクチャ・バージョン4Tでは、この命令は分岐し、分岐先で命令をデコードするのに使用する命令セット・デコーダを選択します。分岐先アドレスはレジスタRmの値です。Tフラグは、レジスタRmの値のビット0で更新されます。

BXはARMコードとTHUMBコード間で分岐するのに使用します。ARMアーキテクチャ4では、これによってTHUMB命令セットをエミュレートできる未定義命令例外が発生します。

この命令は、命令で指定された条件がコンディション・コードのステータスと一致する場合にのみ実行されます。条件は、3.3「コンディション・フィールド」で定義されています。

修飾子

コンディション・コード

注記

THUMBへの遷移：THUMB命令セットに遷移する場合、PCのビット[0]はクリアされ（ゼロにセット）、ビット[31:1]はRmからPCにコピーされます。

ARMへの遷移：ARM命令セットに遷移する場合、PCのビット[0]はクリアされ（ゼロにセット）、ビット[31:1]はRmからPCにコピーされます。Rmのビット[1]がセットされている場合、結果は予測できません。
