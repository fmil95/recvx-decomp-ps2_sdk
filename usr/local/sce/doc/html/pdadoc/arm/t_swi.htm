ARM Architecture Reference Manual

SWI

機能説明

 Software interrupt

アーキテクチャ

 アーキテクチャ v4T のみ

SWI <8_bit_immediate>

説明

SWI命令は、オペレーティング・システムのサービス・コールとして使用します。以下の2とおりの方法で使用できます。

  1. 8ビット・オフセットを使用して、必要なOSサービスを示します。
  2. 8ビット・フィールドを無視して、汎用レジスタで必要なサービスを示します。

SWI例外が発生すると、オペレーティング・システムがそれを処理して、要求されたサービスを提供します（2.5「例外」を参照）。
