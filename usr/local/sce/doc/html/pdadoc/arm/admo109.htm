ARM Architecture Reference Manual

Addressing Mode 1

機能説明

 Rotate right with extend

Rm, RRX

説明

このデータ処理オペランドは、キャリ・フラグを33番目の値として使用して、33ビットの右ローテートを実行するのに使用できます。

レジスタRmの値を1ビットだけ右にシフトし、キャリ・フラグを空きビット位置に挿入して生成されます。

オペレーション

<shifter_operand> = (C Flag Logical_Shift_Left 31) OR 
                    (Rm Logical_Shift_Right 1)
<shifter_carry_out> = Rm[0]

注記

エンコーディング：命令のエンコーディングは、ROR #0に使用される空間に置かれます。

R15の使用：レジスタRmまたはRnとしてR15を指定した場合、使用される値は現在の命令のアドレスに8を加算した値です。

ADC命令：拡張付き右ローテートは、ADC命令で実行できます。
