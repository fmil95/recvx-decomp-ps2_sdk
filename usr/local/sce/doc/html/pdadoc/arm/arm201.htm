ARM Architecture Reference Manual

2.1 データ型

ARMアーキテクチャ・バージョン4のプロセッサは、以下のデータ型をサポートしています。

バイト 8ビット
ハーフワード 16ビット。ハーフワードは2バイト境界に整列していなければなりません。
ワード 32ビット。ワードは4ワード境界に整列していなければなりません。

ARM命令は1ワードです（したがって、4ワード境界に整列します）。

THUMB命令は1ハーフワードです（したがって、2バイト境界に整列します）。

すべてのデータ・オペレーション（例：ADD）は、ワードに対して実行されます。

ロードおよびストア・オペレーションは、メモリとの間でバイト、ハーフワード、およびワードをロードし、ロード時に自動的にバイトまたはハーフワードのゼロ拡張または符号拡張を行います。

符号付きオペランドは2の補数フォーマットです。
