ARM Architecture Reference Manual

CDP

機能説明

 コプロセッサ

アーキテクチャ

 アーキテクチャ v1ではサポートされない

CDP{<cond>} p<cp_num>, <op_code_1>, CRd, CRn, CRm, <opcode_2>

説明

CDP（Coprecessor Data Processing）命令は、<cp#>によって指定されたコプロセッサに、ARMレジスタとメモリから独立したオペレーションを実行するように指示します。命令を実行できることを示すコプロセッサがない場合、未定義命令例外が発生します。

CDPは、たとえば浮動小数点アクセラレータ・コプロセッサの浮動小数点乗算命令など、ARMレジスタやメイン・メモリの値を操作しないコプロセッサ命令を起動するのに使用します。

この命令は、命令で指定された条件がコンディション・コードのステータスと一致する場合にのみ実行されます。条件は、3.3「コンディション・フィールド」で定義されています。
