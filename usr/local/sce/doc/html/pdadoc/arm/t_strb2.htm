ARM Architecture Reference Manual

STRB（2）

機能説明

 Store byte, register offset

アーキテクチャ

 アーキテクチャ v4T のみ

STRB Rd, [Rn, Rm]

説明

この形式のSTRB（Store Register Byte）命令は、汎用レジスタの8ビットのデータをメモリにストアすることができます。このアドレッシング・モードは、ポインタ＋大きなオフセットの算術（MOVイミディエイトを使用してオフセットを設定）や配列の1つの要素にアクセスするのに便利です。

この場合、STRBはレジスタRdの最下位バイトの1バイトをメモリにストアします。メモリ・アドレスは、レジスタRmの値をレジスタRnの値に加算して計算されます。
