ARM Architecture Reference Manual

NEG

機能説明

 Negate register

アーキテクチャ

 アーキテクチャ v4T のみ

MVN Rd, Rm

説明

NEG（Negate）命令はあるレジスタの値をネゲートし、結果を第2のレジスタにストアします。

NEGはゼロからレジスタRmの値を減算し、結果をデスティネーション・レジスタRdにストアします。（結果に基づいて）コンディション・コード・フラグが更新されます。
