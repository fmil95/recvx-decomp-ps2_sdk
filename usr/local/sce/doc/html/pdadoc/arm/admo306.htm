Addressing Mode 3

機能説明

 Register post-indexed
アーキテクチャ アーキテクチャv4 のみ

[Rn], +/- Rm

説明

命令で指定された条件がコンディション・コードのステータスと一致した場合、ベース・レジスタRnの値にインデックス・レジスタRmの値が加算されるか、またはベース・レジスタRnからインデックス・レジスタRmの値が減算されベース・レジスタRnに書き戻されます。条件は、3.3「コンディション・フィールド」で定義されています。

オペレーション

<address> = Rn
if ConditionPassed(<cond>) then
 if U == 1 then

注記

Lビット：ロード命令(L==1)とストア命令(L==0)を区別します。

Sビット：符号付きハーフワード・アクセス(S==1)と符号なしハーフワード・アクセス(S==0)を区別します。

Hビット：ハーフワード・アクセス(H==1)と符号付きバイト・アクセス(H==0)を区別します。

R15の使用：レジスタRmまたはRnとしてR15を指定した場合、結果は予測できません。
