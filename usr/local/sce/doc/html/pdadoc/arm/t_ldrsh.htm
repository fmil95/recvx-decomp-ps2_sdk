ARM Architecture Reference Manual

LDRSH

機能説明

 Load signed halfword, register offset

アーキテクチャ

 アーキテクチャ v4T のみ

LDRSH Rd, [Rn, Rm]

説明

LDRSH（Load Register Signed Halfword）命令は、16ビットの符号付きメモリ・データを汎用レジスタにロードし、その値を操作することができます。このアドレッシング・モードは、ポインタ＋大きなオフセットの算術（MOVイミディエイトを使用してオフセットを設定）や配列の1つの要素にアクセスするのに便利です。

LDRSHは、以下の操作を行います。

  1. メモリからハーフワードをロードし、
  2. このハーフワードを32ビット・ワードに符号拡張して、
  3. これをレジスタRdに書き込みます。

メモリ・アドレスは、レジスタRmの値をレジスタRnの値に加算して計算されます。アドレスがハーフワードに整列していない場合、結果は予測できません。

注記

アラインメント：インプリメンテーションに、システム・コントロール・コプロセッサ（英文リファレンスマニュアル第7章「システム・アーキテクチャとシステム・コントロール・コプロセッサ」を参照）が含まれており、アラインメント・チェックがイネーブルされている場合、ビット[0]!=0のアドレスによって、アラインメント例外が発生します。
