ARM Architecture Reference Manual

SUB（4）

機能説明

 Decrement stack pointer

アーキテクチャ

 アーキテクチャ v4T のみ

SUB SP, SP, #(<7_bit_immediate> * 4)

説明

このSUB（Subtract）命令は、スタックのサイズを増やすのに使用します。

この場合、SPの値から7ビット・イミディエイトの値（0〜127）を減算し、結果をSPに戻します。
