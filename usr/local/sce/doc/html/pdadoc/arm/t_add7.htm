ARM Architecture Reference Manual

ADD（7）

機能説明

 Increment stack pointer

アーキテクチャ

 アーキテクチャ v4T のみ

ADD SP, SP, #(<7_bit_immediate> * 4)

説明

この形式のADD命令は、スタックのサイズを減少させるのに使用します。

この場合、ADDはSPの値と7ビット・イミディエイトの値（0〜127）を加算し、結果をSPに戻します。
