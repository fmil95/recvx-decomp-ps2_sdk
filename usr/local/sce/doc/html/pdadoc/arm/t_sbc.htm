SBC

機能説明

 Subtract with carry (register)

アーキテクチャ

 アーキテクチャ v4T のみ

SBC Rd, Rm

説明

SBC（Subtract with Carry）命令は、64ビットの減算を合成するのに使用します。レジスタ・ペアR0、R1およびR2、R3が64ビット値を保持している場合（R0とR2が最下位ワードを保持）、以下の命令によってR0、R1に64ビットの差が格納されます。
SUB R0,R2
SBC R1,R3

SBCは、レジスタRdの値からレジスタRmの値とNOT（キャリ・フラグ）の値を減算し、レジスタRdに結果をストアします。（結果に基づいて）コンディション・コード・フラグが更新されます。

オペレーション

Rd     = Rd - Rm - NOT(C Flag)
N Flag = Rd[31]
Z Flag = if Rd == 0 then 1 else 0
C Flag = CarryFrom(Rd - Rm - NOT(C Flag))
V Flag = OverflowFrom(Rd - Rm - NOT(C Flag))
