ARM Architecture Reference Manual

Addressing Mode 2

機能説明

 Register post-indexed

[Rn], +/- Rm

説明

このアドレッシング・モードは、ベース・レジスタRnの値からアドレスを計算します。

命令で指定された条件がコンディション・コードのステータスと一致した場合、ベース・レジスタRnの値にインデックス・レジスタRmの値が加算されるか、またはベース・レジスタRnの値からインデックス・レジスタRmの値が減算されてベース・レジスタRnに書き戻されます。条件は、3.3「コンディション・フィールド」で定義されています。

オペレーション

<address> = Rn
if ConditionPassed(<cond>) then
        if U == 1 then
                Rn = Rn + Rm
        else /* U == 0 */
                Rn = Rn - Rm

注記

Bビット：符号なしバイト・アクセス（B==1）とワード・アクセス（B==0）を区別します。

Lビット：ロード(L==1)命令とストア(L==0)命令を区別します。

R15の使用：レジスタRnまたはRmとしてR15を指定した場合、結果は予測できません。

オペランドの制限：RnとRmに同じレジスタを指定した場合、結果は予測できません。
