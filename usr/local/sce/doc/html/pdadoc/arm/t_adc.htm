ARM Architecture Reference Manual

ADC

機能説明

 Add with carry(register)

アーキテクチャ

 アーキテクチャ v4T のみ

ADC Rd, Rm

説明

ADC（Add with Carry）命令は、64ビット加算の合成に使用します。レジスタ・ペアR0、R1およびR2、R3が64ビット値を保持している（R0とR2が最下位ワードを保持）場合、以下の命令によってR0、R1に64ビットの和が格納されます。

ADD R0,R2
ADC R1,R3

命令ADC R0,R0は、R0に対して単１ビットのRotate Left with Extendオペレーション（キャリ・フラグを含む33ビット・ローテート）を実行します。

ADCはレジスタRdの値、キャリ・フラグの値、レジスタRmの値を加算して、レジスタRdに結果をストアします。（結果に基づいて）コンディション・コード・フラグが更新されます。
