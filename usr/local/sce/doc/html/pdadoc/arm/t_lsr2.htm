ARM Architecture Reference Manual

LSR（2）

機能説明

 Logical shift right, register

アーキテクチャ

 アーキテクチャ v4T のみ

LSR Rd, Rs

説明

この形式のLSR（Logical Shift Right）命令は、レジスタの符号なしの値を（レジスタの）2の累乗変数で除算するのに使用します。

この場合、LSR命令はレジスタRdの値をレジスタRsの最下位バイトの値によって論理右シフトし、結果をレジスタRdに戻します。空きビット位置にはゼロが挿入されます。
（結果に基づいて）コンディション・コード・フラグが更新されます。
