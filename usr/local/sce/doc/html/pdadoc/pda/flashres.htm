FLASHREMAPStatus

フラッシュリマップステータスレジスタ

アクセス型 

Read
リセット値
0x00000000

機能
このレジスタは直近のREGRemapレジスタのFLASHVIRビットへの1設定時点での仮想フラッシュメモリ領域設定関連レジスタの内容解析結果を保持する。このレジスタはリードアクセスにより自動的にクリアされる。

ビット詳細
第0ビット
ビット名称：MULTIPLE(MULTIPLE blocks defined with the same virtual area)
リード値：1：複数のブロックが同一仮想ベースアドレスに割り当てられている。
第1ビット
ビット名称：NOACTIVE(NO ACTIVE blocks)
リード値：1：仮想ベースアドレスが１セットも指定されていない。
第2ビット
ビット名称：DABORT(Data ABORT (create a data abort exception))
リード値：1：マップされていない仮想フラッシュメモリ領域へ、または、からの
データアクセスが行われた。
第3ビット
ビット名称：FABORT(Fetch ABORT (create a fetch abort exception))
リード値：1：マップされていない仮想フラッシュメモリ領域からの命令フェッチ
が試みられた。
