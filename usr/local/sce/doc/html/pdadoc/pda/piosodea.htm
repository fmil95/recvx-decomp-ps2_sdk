PIOSetOutpuData

汎用IOポートデータ出力レジスタ

アクセス型 

Write
リセット値
0x00000000

機能
このレジスタにより、各出力ポートに出力する。

ビット詳細
第１ビット
ビット名称：PIO1(PIO 1 set data out)
ビット機能：このビットによって、LEDを制御する。
ライト値：0：(無効)
：1：LEDを消灯する。
第５ビット
ビット名称：PIO5(PIO 5 set data out)
ビット機能：このビットによって、スピーカのスタンバイモードを制御する。
ライト値：0：(無効)
：1：スピーカをスタンバイモードに設定する。
第６ビット
ビット名称：PIO6(PIO 6 set data out)
ビット機能：このビットによって、赤外通信モジュールの出力レベルを切り替える。
ライト値：0：(無効)
：1：赤外通信モジュールを高出力モードにする。
