REGRemap

メモリマップレジスタ

アクセス型 

Read/Write
リセット値
0x0000000

機能
このレジスタはメモリマップ機能を管理するためのビットスイッチ群を内蔵する。

ビット詳細
第0ビット
ビット名称：GENREM(GENeral REdefined Mapping)
リード値：0：メモリマップは「リセット直後」モードである。
：1：メモリマップは「リマップ後」モードである。
ライト値：0：（無効）
：1：メモリマップを「リマップ後」モードに切り替える。
すでに「リマップ後」モードの場合は効果なし。
第1ビット
ビット名称：FLASHVIR(FLASH VIRtual Mapping)
リード値：0：仮想フラッシュメモリ領域マップが無効である。
：1：仮想フラッシュメモリ領域マップが有効である。
ライト値：0：仮想フラッシュメモリ領域マップを無効とする。
：1：仮想フラッシュメモリ領域マップを有効とする。
