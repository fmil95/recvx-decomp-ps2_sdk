RTCControl

リアルタイムクロックコントローラレジスタ

アクセス型 

Read/Write
リセット値
0x00000000

機能
このレジスタは、リアルタイムクロックを制御するためのビットスイッチ群を内蔵する。
ビット詳細
第0ビット
ビット名称：PRGSEL(ProGrammation mode SELector)
ビット機能：カウンタクロック（F_RTC）の周波数を指定する。
ビット値：0：カウントクロック(F_RTC)は1Hz。(通常モード)
：1：カウントクロック(F_RTC)は4KHz。(プログラミングモード)
第1..３ビット
ビット名称：CNTSEL(RTC CouNTer SELection)
ビット機能：リアルタイムクロック内のカウンタを選択する。
リード値：0：秒のカウンタ。
：1：分のカウンタ。
：2：時のカウンタ。
：3：曜日のカウンタ。
：4：日のカウンタ。
：5：月のカウンタ。
：6：年のカウンタ。
：7：カウンタが選択されていない。
