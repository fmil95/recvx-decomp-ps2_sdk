IFStaticControl

赤外通信静的コントロールレジスタ

アクセス型 

Read/Write
リセット値
0x00000002

機能
このレジスタは、赤外通信モジュールを管理するためのビットスイッチ群を内蔵する。
＜タイトル開発者は、省電力対策の為、タイトル終了時に第０ビットを必ず１に設定する。＞

ビット詳細
第０ビット
ビット名称：MODE(Select the IF MODE)
ビット機能：赤外通信モジュールの送受信モードを設定する。
ビット値：0：受信モード。
：1：送信モード。
第１ビット
ビット名称：STDBY(STanDBY)
ビット機能：赤外通信モジュールの動作モードを設定する。
ビット値：0：赤外通信モジュールはアクティブ。
：1：赤外通信モジュールはスタンバイモード。
第２ビット
ビット名称：BGEN(Block GENeraotr enable)
ビット機能：40KHzパルス発振器の動作を設定する。
ビット値：0：40KHzのパルス発生を許可。
：1：40KHzのパルス発生を禁止。
第３ビット
ビット名称：BFLT(Block FiLTer enable)
ビット機能：入力波形フィルタの動作を設定する。
ビット値：0：波形フィルタを許可。
：1：波形フィルタを禁止。
