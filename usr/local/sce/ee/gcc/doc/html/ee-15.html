Reference

This section contains reference information about the various components of the SKY toolchain.

  • To compile C or C++ source code, refer to Compiler.
  • To learn how the Emotion Engine should interface with the operating system, refer to Emotion Engine ABI Summary.
  • To assemble Emotion Engine source code, refer to Emotion Engine Assembler.
  • To assemble DMA tags, GIF tags, VIF instructions, and VU instructions, refer to DVP Assembler.
  • To link with GNU ld , refer to Linker.
  • To debug with GDB, refer to Debugger.
  • To use the SKY simulator, refer to Simulator.
