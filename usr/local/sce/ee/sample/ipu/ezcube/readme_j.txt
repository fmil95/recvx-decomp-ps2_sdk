3次元ポリゴンにIPUを用いてデコードしたテクスチャを張ったサンプル

<サンプルの解説>
	sample ディレクトリの中の pkt/cube のサンプルに
	ipu/ezmovie のサンプルをテクスチャとして張ったものです。
	動作に関してはcubeのサンプルを、動画に関してはezmovieの
	サンプルを参照してください。
	本サンプルではIPUでデコードしたデータをテクスチャバッファに
	DMA転送し、テクスチャとして表示しています。
		
<ファイル>
	main.c
	ldimage.c
	ezcube.c
	ezcube.h
	1_6.ipu
	ezmovie.h

<起動方法>
	% make		：コンパイル
	% make run	：実行

	コンパイル後、以下の方法でも実行可能
	% dsedb
	> run main.elf

<コントローラの操作方法>
	↑,↓ボタン	：x軸回転
	←,→ボタン	：y軸回転
	L1,L2ボタン	：z軸回転
	R1,R2ボタン	：立方体のz軸方向の移動
	△,×ボタン	：カメラのx軸回転
	○,□ボタン	：カメラのy軸回転
	STARTボタン	：リセット
