Clipping サンプル (VU0)

<サンプルの解説>
	このプログラムは、27個の立方体を表示し、バウンダリーボックスを
	用いて表示領域によるオブジェクト単位のクリッピング、ＧＳの描画
	領域によるポリゴン単位のクリッピングを行うことができます。
	バウンダリーボックスのクリッピングを理解しやすいように表示領域
	より狭い範囲にクリッピング領域を設定しています。
	ファークリップまたはニアクリップ面をαブレンディングにより表
	示し、その切替えを行うことも可能です。

<ファイル>
        main.c          : メインプログラム
        vu0.c 	        : 座標変換、透視変換等の行列演算関数
        cube.s          : オブジェクトデータ(立方体)
        torus1.s        : オブジェクトデータ(トーラス データ分割)
        flower.dsm      : テクスチャデータ

<起動方法>
	% make		：コンパイル
	% make run	：実行

	コンパイル後は、以下の方法でも実行可能
	% dsedb
	> run main.elf

<コントローラの操作方法>
	↑↓(方向キー)	: x軸回転(オブジェクト)
	←→		: y軸回転(オブジェクト)
	L1L2ボタン	: z軸回転(オブジェクト)

	△×ボタン	: x軸回転(視点)
	□○ボタン	: y軸回転(視点)
	R1R2ボタン	: z軸移動(視点)
	左スティック    : x軸,y軸移動(視点)

	SELECTボタン	: オブジェクトの切替え(CUBE/TORUS)
	STARTボタン	: オブジェクトクリップのON/OFF
	左ステイック    : 左スティックを押し込むことで、GS描画領域による
			  クリップのON/OFF の切替えを行ないます(*1) 
	右スティック    : 右スティックを押し込むことで、オブジェクトクリッ				  プ面の切替えを行います(NEAR/FAR)

<備考>
	このサンプルは ANALOGモードスイッチを、ONにした状態で実行して
	ください。
	(*1) オブジェクトクリップ OFF の状態で、R2 ボタンを使ってオブ
	ジェクトが画面から大幅にはみ出し、GS の描画領域を超えるくらいに
	します。この状態で左スティックを ON にすると、はみ出したポリゴン
	がクリッピングされます。
