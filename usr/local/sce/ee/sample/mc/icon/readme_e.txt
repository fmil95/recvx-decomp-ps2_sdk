[3D icon tools]
In this directory, the following two tools that can be operated on Linux are
provided to generate 3D icons.

Program to convert 3D icon files between ASCII and binary forms
---------------------------------------------------------------
The contents of existing icon files can be viewed in a text format.
Also, editing output file in a text format is allowed to change the contents
of icon files.
Existing icon samples are provided as sce/ee/sample/mc/basic/slime[1-3].ico.

Texture compression program for 3D icon in the MEMORY CARD
----------------------------------------------------------
In this program, the texture for icons can be compressed in the format
for icon files. 

Refer to Memorycard Library Overview for further information on 
the 3D icon format and the icon.sys format that specify the icons to 
be displayed on the console MEMORY CARD screen.
