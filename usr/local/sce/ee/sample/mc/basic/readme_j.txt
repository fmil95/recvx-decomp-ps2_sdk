メモリーカードライブラリの基本機能確認

<サンプルの解説>

	このプログラムは、メモリーカードライブラリを用いてフォーマット、
	ディレクトリ一覧、icon.sys 生成などを行うものです。

	マルチタップ経由で接続されたメモリーカードに対してアクセスする
	場合はソース内の #define UseMultiTap を１に定義してください。
	

<ファイル>
	main.c
	slime1.ico
	slime2.ico
	slime3.ico


<起動方法>

        % make          ：コンパイル

        コンパイル後、dsedb を起動します。
        % dsedb
        > run main.elf
        を実行します

<コントローラの操作方法>

        メインメニュー：
	方向キー ... メニュー項目のマークを選択します。
	START ボタン ... マークされているメニューを実行します。

	1. get dir
	↑↓ ... エントリのマークを移動します。
	L1/L2 ボタン ... エントリ数が 16 以上のときエントリ一覧を
		スクロールします。
	START ボタン ... ディレクトリエントリがマークされているときのみ
		サブディレクトリの中に入ります。
	○ボタン ... マークされているエントリを選択します。
		選択されているエントリ名がメニューの直下に表示されます。
		ここで選択したメニューは 3. file status の対象になります。
	×ボタン ... マークされているエントリを削除します。
	SELECT ボタン ... get dir メニューを抜けます
			エラーが発生したときも get dir メニューを抜けます。

	2. mkdir + icon ... タイトル名をエントリ名として持つディレクトリを
		作成し、icon.sys を生成、アイコンファイルをコピーします。
		ゲームデータを保存する場合は最低限この作業が必要になります。
		ゲームデータはここで作成したディレクトリの中に置きます。
		ファイル名は任意です。
	←→ ... カーソルを移動させます。
	↑↓ ... エントリ名を修正します。
	×ボタン ... バックスペースを打ちます
	□ボタン ... カーソル上の文字を 0 に変更します。
	△ボタン ... カーソル上の文字を A に変更します。
	○ボタン ... カーソル上の文字を a に変更します。
	R1ボタン ... 空白を挿入します
	START ボタン ... 設定されたタイトル名でディレクトリの初期設定を行
			 います。
	SELECT ボタン ... ディレクトリ生成をキャンセルします。

	3. file status ... 1. get dir メニューで選択されたファイルの
		ステータスを表示、修正します。
	L1,L2 ボタン ... 項目のマークを移動します。
	←→ ... 各項目の中で、カーソルを移動させます。
	↑↓ ... 各項目の値を修正します。
	START ボタン ... 修正内容をファイルに反映させます。
	SELECT ボタン ... 修正内容をキャンセルします。

	attribute: ファイル属性
		1 ... sceMcFileAttrPS1		PS1 データ
		p ... sceMcFileAttrPDAExec	PDA アプリケーション
		d ... SCE_STM_D			ディレクトリ
		c ... SCE_STM_C			コピー禁止
		x ... SCE_STM_X			実行可能
		w ... SCE_STM_W			書き込み可能
		r ... SCE_STM_R			読み出し可能

	file size: 属性で d の部分が x の場合のみファイルサイズを表示

	create: ファイル生成日時

	modify: ファイル修正日時

	4. format ... メモリーカードをフォーマットします。
	○ボタン ... フォーマットを実行します。
	それ以外のボタン ... フォーマットせずにメインメニューに戻ります。

	5. un-format ... メモリーカードを未フォーマット状態にします。
	○ボタン ... アンフォーマットを実行します。アンフォーマットが
		     完了したら、一旦メモリーカードを抜いてください。
	それ以外のボタン ... フォーマットせずにメインメニューに戻ります

	6. rename ... ファイル（ディレクトリ）名を変更します。
	←→ ... カーソルを移動させます。
	↑↓ ... ファイル（ディレクトリ）名を修正します。
	×ボタン ... バックスペースを打ちます
	□ボタン ... カーソル上の文字を 0 に変更します。
	△ボタン ... カーソル上の文字を A に変更します。
	○ボタン ... カーソル上の文字を a に変更します。
	R1ボタン ... 空白を挿入します
	START ボタン ... 修正内容をファイル（ディレクトリ）名に反映させます。
	SELECT ボタン ... 修正内容をキャンセルします。
