PDAライブラリの基本機能確認

<サンプルの解説>

	このプログラムは、PDAライブラリ機能の使い方や関数呼び出しに
	対する反応を確認するためのものです。

	マルチタップ経由で接続されたメモリーカードに対してアクセスする
	場合はソース内の #define UseMultiTap を１に定義してください。
	

<ファイル>
	main.c


<起動方法>

        % make          ：コンパイル

        コンパイル後、dsedb を起動します。
        % dsedb
        > run main.elf
        を実行します

<コントローラの操作方法>

        メインメニュー：
	方向キー ... メニュー項目のマークを選択します。
	START ボタン ... マークされているメニューを実行します。


	1. get dir ... ファイル一覧を表示します(libmc.aの機能)
	↑↓（方向キー）... エントリのマークを移動します。
	L1/L2 ボタン ... エントリ数が 16 以上のときエントリ一覧を
		スクロールします。
	SELECT ボタン ... get dir メニューを抜けます
			エラーが発生したときも get dir メニューを抜けます。


	2. pda info ... sceMcxGetInfo(), sceMcxSetInfo() を使って扱うこと
		の出来る情報を取得／設定します。
	←→↑↓ ... 選択項目を移動させます。
	L1,L2,R1,R2 ボタン ... 選択項目の値を変更します
	START ボタン ... 修正内容をPDAに反映させます。
		(曜日は自動計算されます)
	SELECT ボタン ... 修正内容をキャンセルしてメインメニューに戻ります。

	Running PDA application ... 実行中のアプリケーション番号を表示／
		実行したいアプリケーション番号を設定します。

	Speaker ... スピーカーの使用許可状態を表示／設定します。

	Infrared ... 赤外線送信機能の使用許可状態を表示／設定します。

	Flash write ... PDAアプリケーションからのフラッシュ書き込み
		許可状態を表示／設定します。

	LED on ... LEDの使用許可状態を表示／設定します。

	Serial No ... PDAのシリアル番号を表示します。

	Date ... リアルタイムクロックの内容を表示／設定します。


	3. disp trans ... PDAのファイル転送中表示を ON/OFFします。
	←→↑↓ ... 選択項目を移動させます。
	L1,L2,R1,R2 ボタン ... 選択項目の値を変更します
	START ボタン ... [start]選択中に押すと表示開始します。
			 [stop]選択中に押すと表示を強制終了します。
	SELECT ボタン ... 設定内容をキャンセルします。

	dir ... 転送方向を指定します(起動アプリケーションでは変化なし)

	time ... [stop]実行がない場合、表示を停止するまでの時間を設定します。


	4. read dev ... デバイスエントリに対して読み込みを実行します。
	←→↑↓ ... 選択項目を移動させます。
	L1,L2,R1,R2 ボタン ... 選択項目の値を変更します
	START ボタン ... 設定内容を実行します。
	SELECT ボタン ... 設定内容をキャンセルします。

	device No. ... 呼び出すデバイス番号を指定します。

	param size ... 固定長パラメータ部のバイト数を指定します。

	data size ... 可変長データ部のバイト数を指定します。

	２つあるメモリダンプのうち上段は固定長パラメータ部、下段は
	可変長データ部の内容です。


	5. write dev ... デバイスエントリに対して書き込みを実行します
	←→↑↓ ... 選択項目を移動させます。
	L1,L2,R1,R2 ボタン ... 選択項目の値を変更します
	START ボタン ... 設定内容を実行します。
	SELECT ボタン ... 設定内容をキャンセルします。
	×ボタン ... 可変長データ部の内容をクリアします

	device No. ... 呼び出すデバイス番号を指定します。

	param size ... 固定長パラメータ部のバイト数を指定します。

	data size ... 可変長データ部のバイト数を指定します。

	２つあるメモリダンプのうち上段は固定長パラメータ部、下段は
	可変長データ部の内容です。

	可変長データ部のバッファは 4.read dev, 6.edit mem のバッファと
	共用しており、4.,6.のメニューで実行した結果をこのメニューで利用
	することが出来ます。クリアしたい場合は ×ボタン を押してください。


	6. edit mem ... PDAのメモリ内容を表示／修正します。
	←→↑↓ ... 選択項目を移動させます。
	L1,L2,R1,R2 ボタン ... 選択項目の値を変更します
	(current offset が選択されているときに押すと offset を入力
	し直すことなく別のアドレスの内容を見ることが出来ます)

	START ボタン ... 修正内容をPDAに転送します。
	SELECT ボタン ... 修正内容をキャンセルします。

	current offset ... 現在表示中のメモリダンプの先頭アドレスを
		表示しています。

	offset = ... このメニューに入ったときと、SELECT ボタンを押した
		ときに表示され、メモリダンプの先頭アドレスを変更すると
		きに入力します。

	アクセスするアドレスによっては、PDAがバスエラーを起こしたり
	書き込んだ内容が、そのままの形では読み出せないデータもあります。


	7. edit UIFS ... ユーザーインターフェースステータスを表示／修正
			 します。
	←→↑↓ ... 選択項目を移動させます。
	L1,L2,R1,R2 ボタン ... 選択項目の値を変更します
	START ボタン ... 修正内容をPDAに転送します。
	SELECT ボタン ... 修正内容をキャンセルします。

	Alarm ... アラーム時刻およびアラームON/OFF 状態を表示／設定します。

	key lock ... キーロック状態を表示／設定します。

	Volume ... スピーカー音量を表示／設定します。

	Area code ... PDAの仕向地を表示します。

	Rtc set ... リアルタイムクロック設定済みフラグを表示／設定します。

	Font addr ... LCD表示に使っているフォント格納アドレスを表示します。


	8. LED表示 ... LEDを ON/OFF します。	
	←→ ... on/off を選択します。
	START ボタン ... 選択内容をPDAに転送します。
	SELECT ボタン ... 選択内容をキャンセルします。
